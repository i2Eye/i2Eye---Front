# i2Eye frontend

1. Check if node is installed in the system.
##### Run `node -v`

- If node is not installed, go to https://nodejs.org/en/download/


2. ##### Run `npm install` to install the list of packages used in the project.


3. ##### Run `npm start` to start the project. <br/>
The project will run in http://localhost:3000. The page will reload if you make edits. You will also see any lint errors in the console.
